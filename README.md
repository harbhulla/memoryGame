# 🧠 Memory Game

A memory game built with React and Vite using the PokéAPI. Try to click each Pokémon only once!

## 🚀 Live Demo

👉 [Play it here]([https://memorygame.vercel.app](https://memory-game-kwuu.vercel.app/))

## 🛠️ Tech Stack

- React
- Vite
- PokéAPI
- CSS Grid/Flexbox
- Deployed on Vercel
